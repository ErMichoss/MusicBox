# Music Box

Un mod para R.E.P.O. (Unity) con soporte de multijugador usando Photon (PUN) que permite reproducir música personalizada dentro del juego mediante una interfaz simple.

## Instalación

1. Descarga el source code y compilalo con Visual Studio 2022
2. Crea una carpeta dentro del directorio plugins de BepInEx
3. Crea la carpeta para las canciones: /MusicBox/songs
4. Dentro de esa carpeta metes las canciones

## Uso
#### Abrir el Menu
· Pulsa la tecla "M" para abrir cerra el menú.
#### Controles
· Seleccionar canción --> Click en la lista
· Play --> Reproducir Canción
· Stop --> Detener reproduccion
· Cerrar --> Cerrar Menú
